# 🏨 Hostel Management System

A comprehensive web-based hostel management platform developed using PHP, MySQL, HTML, CSS, and JavaScript to streamline hostel operations including student registration, room allocation, mess management, complaint handling, blog posting, and hostel administration.

The system provides a centralized solution for managing hostel activities, reducing manual paperwork, and improving communication between students and hostel managers.

---

## 📌 Project Overview

Hostel Management System is designed to automate hostel administration processes and improve the overall hostel management experience. The platform allows students to register, apply for hostel rooms, manage mess services, submit complaints, and access hostel-related information through a user-friendly web interface.

Hostel managers and administrators can efficiently manage room allocations, track vacancies, handle student requests, monitor mess services, and oversee hostel operations through dedicated management modules.

## 🏗️ System Workflow

1. Student registers and logs into the system
2. Student submits hostel room application
3. Hostel manager reviews applications
4. Room allocation is processed
5. Student accesses hostel services
6. Mess registration and allocation are managed
7. Complaints are submitted and resolved
8. Hostel operations are monitored through admin modules

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Manikumarveldurthi/hostelmanagementsystemptu.git
```

### Navigate to Project Directory

```bash
cd hostelmanagementsystemptu
```

### Install Dependencies

```bash
composer install
```

### Configure Database

* Create a MySQL database
* Import the database dump
* Update database credentials in project configuration files

### Run Application

* Start Apache and MySQL using XAMPP
* Place the project folder inside the `htdocs` directory
* Open the browser and visit:

```text
http://localhost/hostelmanagementsystemptu
```
